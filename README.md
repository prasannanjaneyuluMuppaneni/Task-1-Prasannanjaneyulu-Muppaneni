# Number Guessing Game

This is a simple Java project. In this game, the computer generates a random number between 1 and 100 and the user has to guess it.

## Features
- Random number generation
- Maximum 7 attempts
- Score calculation
- High and Low hints
- Exception handling
- Play again option

## Tools Used
- Java
- Scanner Class
- Random Class

## How to Run

Compile the program:

```bash
javac DecodeLabs_Java_P1.java

java DecodeLabs_Java_P1
